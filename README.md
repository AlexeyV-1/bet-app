# bet-app
- Простое приложение, которое имитирует азартную игру:
1. Вы можете выставлять ставку, которая ограничивается вашим балансом, либо держать минимум для игры - 10 очков;
2. При выигрыше на экране показывается поздравительная строка с разными надписями, также положительные цвета - зеленый. При проигрыше появляются соответствующие реплики, также видны красные цвета;
3. Если баланс слишком низкий, а вы поставили ставку, которая привышает ваш баланс, на экране появится соответствующее уведомление о невозможности продолжить игру;

- Возможно изменить стартовое количество очков для игры в файле state.js, параметр - balance;
- Возможно изменить шанс выигрыша (WIN_PROBABILITY), на сколько раз будет умножен выигрыш (WIN_MULTIPLIER) в файле config.js;
